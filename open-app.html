AgroMart Frontend Launcher

Option 1: Run directly in Command Prompt

Open a Command Prompt window and run these commands:

cd /d D:\AgroMart\frontend
npm run dev

Option 2: Run with administrative PowerShell

Open PowerShell as Administrator and run:

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
cd D:\AgroMart\frontend
npm run dev

Note: The application should be running at http://localhost:5173 once started.
